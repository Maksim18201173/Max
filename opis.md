1. Верхняя часть страницы (Шапка)
1.1 Логотип сайта
Расположение: В левом верхнем углу

Внешний вид: Слово "Phonelectro", где первые четыре буквы "Phon" выполнены синим цветом, а часть "electro" - оранжевым

Функционал: При клике на логотип происходит переход на главную страницу (в текущей версии работает как ссылка на текущую страницу)

1.2 Контактный телефон
Расположение: В правом верхнем углу

Внешний вид: Рядом с номером телефона изображена иконка телефона

Текст: "8-800-123-45-67"

Размер текста: Стандартный

Цвет: Темно-серый

Функционал: Статический элемент для отображения контактной информации

1.3 Основное меню навигации
Расположение: Под логотипом, в левой части

Формат: Горизонтальный список

Пункты меню:

Главная - с иконкой домика

Смартфоны - с иконкой мобильного телефона

Ноутбуки - с иконкой ноутбука

Внешний вид пунктов: Темно-серый текст жирного начертания

Интерактивность: При наведении курсора цвет текста меняется на синий

Функционал: Предназначены для перехода между разделами сайта (в текущей версии ссылки не активны)

1.4 Поисковая строка
Расположение: По центру навигационной панели

Состав:

Поле для ввода: С подсказкой "Поиск товаров..."

Кнопка поиска: С иконкой лупы

Внешний вид:

Поле ввода: Светло-серая рамка, скругленные левые углы

Кнопка: Синий фон, белая иконка лупы, скругленные правые углы

Интерактивность: При наведении на кнопку курсор меняет вид

Функционал:

При нажатии на кнопку или клавишу Enter проверяется, не пустое ли поле

Если поле пустое - появляется сообщение "Введите поисковый запрос"

Если текст введен - появляется сообщение "Поиск: [введенный текст]"

В полноценной версии должна осуществлять поиск товаров

1.5 Значки пользовательских действий
Расположение: В правом краю навигационной панели

Элементы:

Профиль пользователя - иконка силуэта человека

Избранное - иконка сердца

Корзина покупок - иконка тележки для покупок

Интерактивность: При наведении цвет меняется на синий

4. Адаптивность (подстройка под разные устройства)
4.1 Для планшетов (экраны ≤ 992px):
Сетки категорий и товаров меняются с 4 на 3 колонки

Поле поиска немного уменьшается

4.2 Для небольших планшетов (экраны ≤ 768px):
Сетки меняются с 3 на 2 колонки

Элементы шапки перестраиваются вертикально

Поле поиска растягивается на всю доступную ширину

4.3 Для мобильных телефонов (экраны ≤ 576px):
Сетки меняются с 2 на 1 колонку

Верхняя часть шапки становится вертикальной

Уменьшаются отступы между элементами меню

Уменьшается размер заголовка баннера
