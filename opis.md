5. АДАПТИВНЫЙ ДИЗАЙН (MEDIA QUERIES)
5.1 ДЛЯ ЭКРАНОВ ≤ 992px
Сетки категорий и товаров: с 4 на 3 колонки

Поле поиска: ширина уменьшается до 200px

5.2 ДЛЯ ЭКРАНОВ ≤ 768px
Сетки категорий и товаров: с 3 на 2 колонки

Шапка: элементы располагаются вертикально

Навигация: переносится на новую строку

Поиск: растягивается на всю ширину

Пользовательские действия: выравниваются по правому краю

5.3 ДЛЯ ЭКРАНОВ ≤ 576px
Сетки категорий и товаров: с 2 на 1 колонку

Верхняя часть шапки: элементы располагаются вертикально

Контакты: добавляется отступ сверху 10px

Навигация: уменьшаются отступы между пунктами

Баннер: уменьшается паддинг и размер заголовка (24px)

Заголовок баннера: уменьшается до 24px
