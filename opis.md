4. Компоненты интерфейса
Кнопки:
Размер: 10px вертикальный отступ, 20px горизонтальный

Скругление: 3px (небольшое)

Цвет фона: #0066cc (синий)

Цвет текста: #ffffff (белый)

Ховер-эффект: фон меняется на #0055aa (более темный синий)

Transition: плавное изменение фона за 0.3s

Карточки товаров:
Размер: адаптивный (резиновая сетка)

Фон: белый

Граница: 1px solid #dddddd

Скругление: 5px

Внутренние отступы: 15px

Ховер-эффект: тень box-shadow: 0 2px 8px rgba(0,0,0,0.1)

Transition: плавное появление тени за 0.3s

Карточки категорий:
Размер: адаптивный

Фон: белый

Граница: 1px solid #dddddd

Скругление: 5px

Внутренние отступы: 20px

Выравнивание: по центру

Иконки категорий: 40px размер, синий цвет

Формы:
Поле поиска:

Высота: примерно 36px

Граница: 1px solid #dddddd

Скругление: левая сторона 3px

Кнопка поиска: правая сторона 3px

Цвет кнопки: синий #0066cc

5. Сетка и отступы
Брейкпоинты:
Десктоп: ≥ 992px - 4 колонки

Планшет: 768px-992px - 3 колонки

Мобильный (горизонтальный): 576px-768px - 2 колонки

Мобильный (вертикальный): < 576px - 1 колонка

Контейнер:
Максимальная ширина: 1200px

Внутренние отступы: 0 15px

Выравнивание: по центру

Вертикальные отступы:
Между секциями: 30px-40px

Внутри карточек: 10px-20px

Подвал: 30px сверху, 15px снизу

6. Эффекты и анимации
Ховер-эффекты:
Ссылки навигации: изменение цвета текста с #333 на #0066cc

Карточки товаров: появление тени

Кнопки: изменение фона с #0066cc на #0055aa

Ссылки в подвале: изменение цвета с #ccc на #fff

Анимации:
Плавные переходы:

Изменение фона кнопок (0.3s)

Появление тени карточек (0.3s)

7. Особенности реализации
Адаптивность:
Mobile-first подход: стили для мобильных, затем медиа-запросы для больших экранов

Гибкие изображения: max-width: 100%

Адаптивная сетка: CSS Grid с repeat(auto-fit, minmax())

Доступность:
Alt-текст для всех изображений

Семантическая разметка (header, main, footer, nav)

Интерактивные элементы с указанием курсора

Производительность:
Внешние ресурсы: Font Awesome через CDN

Изображения: через placeholder (в прототипе)

Код: минимизированные встроенные стили и скрипты
